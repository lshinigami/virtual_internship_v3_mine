Приветствую вас!
Виртуальная Стажировка (Java) это большой и интересный проект.

На разработку этого проекта автор курса потратил вот уже 2 года.

За это время было создано 220 основных проектных заданий + 40 бонусных.

Проект охватил следующие темы:

1. Build Tools, Gradle
2. IoC (Inversion of Control)
3. Unit testing: JUnit + Mockito
4. Lombock и работа с аннотациями
5. Логирование
6. REST API + Postman
7. Основы сетевого взаимодействия, HTTP
8. Anticorruption layer, валидация данных
9. Spring, тспользование файлов настроек
10. Domain Driven Design (DDD)
11. Деление кода приложения: техническое vs DDD
12. Декомпозиция фуекционала на подсистемы и дизайн взаимодействия подсистем в приложении.
13. SQL мини курс
14. JDBC + ORM + JPA + Spring JPA
15. Liquibase управление миграциями реляционной базы данных.
16. Integration and acceptance testing
17. Java Web Server
18. Servlet API specification
19. Деплоймент приложения (war, jar, fatJar)
20. Spring Boot
21. Многопоточность и нагрузочное тестирование
22. Hexagonal Architecture
23. Clean Code + refactoring
24. REST API версионирование
25. MVC, web UI
26. Background jobs
27. CI/CD
28. Git, GitHub, branching
29. Agile, sprints
30. Design Patterns: DTO, Builder, Factory, Strategy
31. SRP (Single Responsibility Principle)
32. DRY (Do not Repeat Yourself)
33. Stream API
34. Декомпозиция и наследование
35. XML, JSON
36. H2, MySQL (реляционные базы данных)

Бонус темы:
- Observability (логи + метрики + трейсинг)
- 12 factory app
- Docker, docker compose, контейнеризация
- Брокеры сообщений
- Синхронная и асинхронная интеграции
- Микро сервисная архитектура
